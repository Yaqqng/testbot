# Telegram-бот для генерации случайных цифр

## Что делает бот
- Команда `/start` показывает подсказку.
- Команда `/digits` генерирует 8 случайных цифр.
- Команда `/digits N` генерирует `N` случайных цифр (от 1 до 128).

## Запуск
1. Создай бота через [@BotFather](https://t.me/BotFather) и получи токен.
2. Установи зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Установи токен в переменную окружения:
   ```bash
   export TELEGRAM_BOT_TOKEN="твой_токен"
   ```
4. Запусти бота:
   ```bash
   python bot.py
   ```

## Пример
- `/digits` → `Случайные цифры: 58390124`
- `/digits 12` → `Случайные цифры: 019347552884`
